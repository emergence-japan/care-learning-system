# トラック仕様書: 監査用レポート機能の刷新 (audit_report_refresh)

## 1. 概要
実地指導（監査）において、施設管理者が行政にそのまま提出できるクオリティの「研修実施状況報告書」および「受講証明書」をPDF形式で出力する機能を実装する。既存の年間スケジュール設定と連動し、コンプライアンス遵守を視覚的に証明可能にする。

## 2. 機能要件

### 2.1 PDFレポート生成エンジン
- サーバーサイド（または信頼性の高いクライアントサイドライブラリ）によるPDF生成。
- レイアウトの固定化（ブラウザ依存の排除）。
- 施設名、出力日、対象年度の自動印字。

### 2.2 レポート種類と項目
- **研修実施サマリー（施設全体）**
  - 年度内の全法定研修の実施状況一覧。
  - 各科目の実施日、受講人数、修了率。
  - 設定済みの「年間スケジュール」との対比。
- **スタッフ別受講記録（個人）**
  - 特定スタッフの全科目修了状況。
  - 各科目の修了日およびテストスコア。
  - 受講証明書としてのフォーマット。
- **年間スケジュール出力**
  - 施設管理者が設定した研修計画のガントチャートまたはカレンダー形式出力。

### 2.3 ユーザーインターフェース (UI)
- **施設管理者ダッシュボード**
  - 「一括レポート出力」ボタンの配置。
  - 現在の進捗状況をクイックにPDF化する機能。
- **レポート管理専用画面（新設）**
  - 年度（2024年度、2025年度など）の選択。
  - 出力対象（全スタッフ、特定部署、特定科目）のフィルタリング。
  - 複数レポート（全体＋個人全員分など）の一括生成・ZIPダウンロード機能。

## 3. 非機能要件
- **正確性:** データベース上の受講完了フラグおよびテストスコアと100%一致すること。
- **操作性:** 大人数（例：100名以上）のスタッフがいる施設でも、タイムアウトせずに一括生成できること。
- **デザイン:** 行政提出用として相応しい、清潔感のあるフォーマルな帳票レイアウト。

## 4. 完了定義 (Acceptance Criteria)
- [ ] 施設管理者がダッシュボードから、当年度の「研修実施サマリー」をPDFでダウンロードできる。
- [ ] 新設のレポート管理画面から、特定の年度とスタッフを指定して「受講記録」をPDF出力できる。
- [ ] 出力されたPDFに、施設で設定済みの「年間スケジュール」が正しく反映されている。
- [ ] 10名以上のスタッフデータを一括でPDF生成（またはZIP化）してダウンロードできる。

## 5. スコープ外 (Out of Scope)
- 研修コンテンツ（スライド内容）自体のPDF出力。
- 外部システム（行政の電子申請システム等）への直接連携。
