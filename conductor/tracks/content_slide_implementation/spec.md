# Track Specification: Slide-based Learning Content

## 概要
従来の動画視聴型から、スライド（HTML/JS）を主体とし、補助的に動画を視聴できるハイブリッド型の学習コンテンツ構成に変更する。

## 変更点
- **スキーマ拡張**: `Course` モデルに `Slide` モデルを紐付け。
- **UI/UXの刷新**: スライドプレイヤーの実装（進捗バー、スライド切り替え、補助動画再生）。
- **コンテンツ実装**: 実践的な研修コンテンツを1つ完成させる。

## 完了条件
- [ ] `Slide` モデルが定義され、マイグレーションが完了していること。
- [ ] 1つのコースに対して複数のスライドを表示・ナビゲーションできること。
- [ ] スライドごとに補助動画を再生できる機能があること。
- [ ] 全スライド完了後に理解度テストへ遷移すること。
- [ ] サンプルコンテンツ（虐待防止研修など）が投入されていること。
