# Track: 施設管理者ダッシュボードの機能強化と詳細UI改善

## 概要
施設管理者が日常業務および監査対応において、より効率的かつ確実に操作・確認を行えるよう、機能の追加とUIの微調整を行います。

## 実施事項

### 1. 監査用情報のPDF出力機能
- [x] 研修計画（年間スケジュール）およびスタッフの受講完了状況を、公的な監査（実地指導等）に提出可能なフォーマットでPDF出力する機能を実装。
- [x] 「PDFで書き出し」ボタンの配置。
- [x] ブラウザの印刷用スタイル（Print CSS）の最適化。

### 2. スタッフ管理UIのツールチップ追加
- [x] スタッフ一覧にある「鍵マーク（パスワード変更）」および「ゴミ箱マーク（削除）」に、カーソルを合わせた際に機能説明が表示されるポップアップ（Tooltip）を実装。
- [x] 「未受講科目を確認（本マーク）」にも同様の説明を追加。

## 技術的検討事項
- PDF生成: `jspdf` / `html2canvas` または、サーバーサイド生成の検討。
- ツールチップ: Radix UI (shadcn/ui) の Tooltip コンポーネントの導入。
