# Track: システム管理者（Super Admin）ダッシュボードの刷新

## 概要
システム全体の管理画面を使いやすく、かつ実務に即した安全な設計へと刷新します。

## 重要要件・運用ポリシー（2026/02/09 指示）

### 1. 法人管理の安全性（データ保持優先）
- **法人削除の制限**: システム管理者は法人を安易に削除（DB物理削除）せず、「ステータスを停止（ロック）」に変更することで、契約終了後のアクセスを制限する運用とする。
- **理由**: 法定研修の履歴保持義務（監査対応）および誤操作による大規模なデータ消失を防ぐため。

### 2. データ修正機能
- **誤字脱字の修正**: 登録済みの以下の項目について、後から編集・修正できる機能を実装する。
    - 人名（ユーザー名）
    - 施設名
    - （その他、入力ミスが発生しやすいマスター情報）

- [ ] 各種情報のインライン編集（Edit）機能の実装。
