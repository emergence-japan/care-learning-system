# Track: システム管理者（Super Admin）ダッシュボードの刷新

## 概要
システム全体の管理画面を使いやすく、かつ実務に即した安全な設計へと刷新します。

## 実施済み事項（2026/02/09 完了）

### 1. 法人・施設のステータス管理機能
- [x] `isActive` フィールドを `Corporation` および `Facility` モデルに追加。
- [x] システム管理者から「利用停止 / 再開」を切り替える機能を実装。
- [x] 利用停止時の制限ロジックを実装。
    - 受講者（STAFF）: ログインを完全に禁止。
    - 管理者（HQ / ADMIN）: ログインは可能だが「閲覧・PDF出力のみ」の制限モード（監査対応用）となり、情報の新規登録や変更は不可。

### 2. データ修正機能（誤植対応）
- [x] 以下の項目の編集・修正機能を実装。
    - ユーザー氏名、ログインID、パスワード
    - 法人名
    - 施設名、定員数
- [x] 各管理権限（SUPER_ADMIN, HQ, ADMIN）に応じた編集範囲の制限。

### 3. 一覧画面のUI/UX高度化
- [x] 組織・施設一覧をコンパクトな「1行リスト形式」に刷新。
- [x] 精密なレイアウト調整により、法人と施設の各情報の縦ラインを垂直に統一。
- [x] インタラクティブな詳細ポップアップの実装。
    - スタッフ数クリック: 氏名リストと各人の進捗率（分数付き）を表示。
    - 研修数クリック: 割当済み研修科目リストと施設内完了率を表示。
- [x] 進捗率（％）の自動計算と表示の実装。

### 4. システム安定性の向上
- [x] 研修（Course）削除時の外部キー制約エラーを解消（割当情報の連鎖削除）。
- [x] 削除操作時の誤操作防止用「確認ダイアログ」を全ての削除アクションに導入。
- [x] 全UIの日本語化。

## 完了のまとめ
本改修により、システム管理者は「横スクロールなしの1画面」で、全法人の契約状況、各施設の業態、研修の進捗実数（誰が何科目終えたか）までを即座に把握・操作できるようになりました。
