# Track Specification: プロジェクトの基盤構築と認証・ダッシュボードの基本実装

## 概要
Next.js, TypeScript, Tailwind CSS, Prisma を使用したプロジェクトの土台を作り、ユーザー認証（ログイン）と、介護職員が自分の受講状況を確認できるダッシュボードのプロトタイプを構築します。

## 主要機能
- **プロジェクト初期化:** Next.js プロジェクトのセットアップと必要なライブラリの導入。
- **データベース設計:** ユーザー、研修、受講実績の基本スキーマ定義。
- **認証機能:** 職員用ログイン画面と認証ロジックの実装。
- **スタッフ用ダッシュボード:** ログイン後、未受講・受講済みの研修が一覧表示される画面の構築。

## 成功基準
- プロジェクトがビルド可能であること。
- ユーザーがログインし、マイページ（ダッシュボード）を表示できること。
- スマートフォン表示において、大きなボタン等のモバイルフレンドリーなUIが実現されていること。
